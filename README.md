# 🖥️ Gestión de Peluquería - Aplicación de Escritorio

Este repositorio contiene la **Aplicación de Escritorio** de la suite completa para la gestión de peluquerías. Está diseñada principalmente para el uso del personal administrativo y gestores del negocio en entornos locales de recepción.

---

## 🚀 Stack Tecnológico

* **Lenguaje:** C# (.NET)
* **Interfaz Gráfica:** Windows Forms (WinForms)
* **IDE Recomendado:** Visual Studio 2022
* **Consumo de API:** Integración con la API REST de la suite (Backend en Spring Boot).

---

## 🛠️ Características Principales

* **Interfaz Nativa de Escritorio:** Panel de control optimizado para la gestión de clientes, servicios y citas diarias desde recepción.
* **Control de Horarios:** Al estar pensada para un Grado de Peluquería y Belleza, permite bloquear días y horas de forma libre en caso de tener semanas de exámenes, vacaciones, etc.
* **Reportes Visuales:** Generación de resúmenes informativos básicos sobre la facturación y los servicios más solicitados.
* **Jerarquía de usuarios** Distintos usarios con diferentes permisos en función de si es un profesor o alumno.

---
